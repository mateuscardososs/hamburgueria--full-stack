# Hambúrguer Explosivos

Bem-vindo ao **Hambúrguer Explosivos**! Um projeto de site para um restaurante, oferecendo a melhor experiência de hambúrguer com um design responsivo e moderno.

---

## 🖥️ Tecnologias Utilizadas

- **HTML5**: Estrutura do site.
- **CSS3**: Estilização e layout.
- **JavaScript**: Interatividade.
- **Bootstrap Icons**: Ícones para mídias sociais e elementos visuais.
- **Fontes do Google**: `Roboto` para tipografia.

---

## 🌟 Funcionalidades

- **Menu de navegação fixo**: Facilita o acesso às seções.
- **Seções principais**:
  - Home: Apresentação do restaurante.
  - Sobre nós: Informações sobre o restaurante e sua missão.
  - Cardápio: Exibição dos itens disponíveis com preços.
  - Contatos: Informações para entrar em contato.
- **Responsividade**: Adaptado para diferentes dispositivos.
- **Interatividade**: Botões e hover effects.
